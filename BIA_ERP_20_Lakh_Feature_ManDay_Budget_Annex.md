# 4. Estimation Assumptions

- One man-day represents up to eight productive working hours.
- Estimates assume reuse of proven technical components and standardized screens/workflows.
- The client will provide final rules, samples and decisions; policy analysis or legal interpretation is excluded.
- Only the listed basic features are included. Each new workflow, formula, integration or report requires impact assessment.
- Integration estimates assume usable English documentation, test credentials and one client/provider certification cycle.
- Data migration covers opening/current data in agreed templates, not cleansing or complete historical transactions.
- Client UAT time and third-party/vendor waiting time are not counted as supplier man-days.
- Man-day estimates may move between features by mutual agreement without changing the fixed total scope, price or delivery obligations.

# 5. Estimate Qualification

The BDT 20,00,000 price is intentionally optimized for a basic implementation. It is commercially workable only when the signed scope remains standardized and capped. If discovery confirms advanced accounting, statutory payroll, multiple workflows, undocumented integrations, large migration volume or additional reports, those items must be prioritized for a later phase or handled through an approved change order.
